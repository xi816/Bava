# Bava

## Description
Bava is a programming language made in Python.

## Run
To run your **Bava** program, do these steps:
  - Check that file ends with `.bava`
  - Write `python3 path/to/the/interpreter/main.py -d:0 -c:cache/folder path/to/your/bava/file.bava`
P.S. Your programs are staying at `run/file-tree.bava.ast`.

## Debug
To debug a **Bava** program, write `python3 path/to/the/interpreter/main.py -d:1 -c:cache/folder path/to/your/file.bava`
